# WP Maintenance

Simple WordPress maintenance mode plugin.

## Features

- Enable/disable maintenance mode from an admin settings page.
- Custom meta title and meta description.
- Custom HTML, CSS, and JavaScript for the maintenance screen.
- Optional Google Analytics ID support.
- Sends a `503 Service Unavailable` status while maintenance mode is enabled.
- Adds no-cache and noindex headers for the maintenance response.
- Logged-in users can still access the site normally.
